# Wanderlust Rooms - Full Stack Application

This is a complete full-stack room booking platform for tourist destinations across India.

## Quick Start

### Backend Setup
```bash
cd backend
npm install
npm run seed  # Seed 15 cities with 90 rooms
npm run dev   # Start on http://localhost:5000
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev   # Start on http://localhost:3000
```

## Documentation

- **Full README**: See [README.md](README.md)
- **Backend**: `backend/` folder with Express.js API
- **Frontend**: `frontend/` folder with Next.js UI

## Key Features

✅ Room booking across 15 Indian cities
✅ User authentication with OTP
✅ JWT-based security
✅ Booking management
✅ Complaints system
✅ Admin dashboard
✅ Modern Airbnb-like UI

## Tech Stack

- **Frontend**: Next.js, TailwindCSS, Framer Motion
- **Backend**: Express.js, MongoDB
- **Auth**: JWT + OTP
